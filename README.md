# E-Commerce-Page-Salima

A simple e-commerce page . This project demonstrates a basic online shopping experience with product display, shopping cart functionality, and responsive design.

## Features

- **Product Display**: Showcases a grid of retro sneakers with images, titles, and prices.
- **Shopping Cart**: Add items to cart, adjust quantities, remove items, and view total price.
- **Responsive Design**: Optimized for desktop and mobile devices.
- **Interactive UI**: Hover effects on product images, cart icon with item count, and smooth cart animations.
- **Buy Now Functionality**: Simulates a purchase with a confirmation alert.

## Technologies Used

- **HTML5**: Structure of the web page.
- **CSS3**: Styling and responsive layout using CSS Grid and Flexbox.
- **JavaScript (ES6)**: Dynamic product rendering, cart management, and event handling.

## Usage

- Browse the products in the shop section.
- Click the shopping bag icon on any product to add it to the cart.
- Click the cart icon in the header to view your cart.
- In the cart, adjust quantities using + and - buttons, or remove items.
- Click "Buy Now" to complete the purchase .
